Projects

Mall Customer data Segmentation using K-Means Clustering

Clemson University
The purpose of this project is to identify the distinct user groups based on the data. Mall Consumer Data is used, and it is a fascinating dataset that contains fictitious customer information, and From the perspective of the supermarket owner, it is always difficult to distinguish between the users. After researching several Machine Learning Algorithms, out of which K-Means clustering is a better algorithm to solve this problem and comprehend consumer data. The K-Means technique is an excellent clustering algorithm. Almost every cluster has the same density. In terms of computing cost, it's also quick and efficient.


Evaluation of Skull-Strip & deidentification algorithms using supervised learning

Clemson University
The purpose of this project was to develop a data mining algorithm to identify any brain feature loss and any facial features that exist that concern the privacy of the user in the MRI scans. So we propose a system that could use supervised learning and deep learning techniques and predict if there is any brain feature loss that has happened and if any facial features exist in the MRI Scan. A deep convolutional model is proposed that helps solve the problem statement. We achieved an accuracy of 96% and a recall of 1.0 with the proposed model.
